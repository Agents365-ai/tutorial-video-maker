# Tutorial Outline: How to Use LM Studio for Web Search

## Overview

**Duration:** 8-10 minutes
**Target Audience:** Developers and AI enthusiasts who want to run local LLMs with web search capability
**Prerequisites:** LM Studio installed, basic familiarity with LLMs

### Chapter 1: Introduction
- **Video:** `assets/01_intro.mp4`
- **Duration:** 1 min
- **Key Points:**
  - [ ] Welcome and overview of LM Studio web search

  - [ ] Prerequisites: LM Studio 0.3+ installed

### Chapter 2: Enabling Web Search
- **Video:** `assets/02_enable_web_search.mp4`
- **Duration:** 2 min
- **Key Points:**
  - [ ] Open LM Studio settings
  - [ ] Navigate to the Web Search tab
  - [ ] Enable web search feature

- **Video:** `assets/03_load_model.mp4`

## Notes

- Test with current events questions to verify web search is working
- Some models handle web search better than others
- Consider privacy implications when enabling web search
